📋 LaTeX kompilieren

📦

Lokal kompilieren

Klicken Sie auf „.tex downloaden“, um die LaTeX-Datei herunterzuladen.

💻 Option 1: Lokaler Compiler

Kompilieren Sie mit Ihrer lokalen LaTeX-Installation:

pdflatex brief.tex

Voraussetzung: TeX Live, MiKTeX oder MacTeX

🌐 Option 2: Online-Dienste

Laden Sie die .tex-Datei in einen dieser Dienste hoch:

  • Overleaf (kostenlos, Account erforderlich)
  • Papeeria (kostenlos)
  • LaTeX Base (kein Account)

🚀 Option 3: Cloud-Compiler API

Verwenden Sie einen dieser kostenlosen APIs:

  • latex.ytotech.com
  • LaTeX.Online

Diese erfordern keine Installation, senden aber Daten an externe Server.

⚠️ Hinweis: Eine vollständig browser-basierte Lösung ohne externe Dienste ist technisch möglich (via SwiftLaTeX WebAssembly), aber die CDN-Distribution ist aktuell nicht zuverlässig verfügbar. Für maximale Datensouveränität verwenden Sie bitte einen lokalen Compiler.
